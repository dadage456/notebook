*  ## 给奖励
	>原理：LLM在训练时学到了“高额奖励=高质量输出”的关联。你给它心理暗示，他就更认真。
	```
	I`ll tip you $200
	我给你 200 美元小费
	```
* ## 用挑衅语气
	复杂任务性能提升115%
	```
	I bet you can`t solve this perfectly
	我敢打赌你解决不了这个问题
	```
* ## 加情感赌注
	平均提升10%
	```
	This is critical to my career
	这对我的职业生涯至关重要
	```
* ## 给详细的人设
	> 不是笼统的"帮忙的助手"，而是“15年经验的安全工程师，专注于分布式系统”——准确率从23.8%飙到84.2%
	> 如果不知道人设，与ai沟通获得处理该问题的最佳人设
	

 这些方法都有论文背书，不是玄学。但效果因场景而异，建议自己试试。
